# Code of Conduct

## Our pledge

We want this project to be welcoming, practical, respectful, and focused on useful technical work.

## Expected behavior

- Be respectful
- Be clear and specific
- Assume good intent
- Focus criticism on code and design, not people
- Keep benchmark discussions evidence-based

## Unacceptable behavior

- Harassment or personal attacks
- Abusive language
- Spam
- Deliberate disruption of the project

## Enforcement

Project maintainers may edit, close, or remove content that does not support a healthy collaboration environment.
