# Security Policy

## Supported versions

Use the latest tagged release whenever possible.

## Reporting a vulnerability

Please do not open public issues for suspected security problems.

Instead, contact the maintainer privately with:
- affected version
- environment details
- reproduction steps
- proof-of-concept if available
- impact description

## Scope

This is a local desktop benchmark tool. Security concerns may still matter if they involve:
- arbitrary file writes
- unsafe parsing
- unsafe plugin or script loading
- remote code execution
- unexpected elevation of privilege
